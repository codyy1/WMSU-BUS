# Images Directory

This directory is used to store image files for the WMSU Transport application.

## Supported Formats:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- SVG (.svg)

## Usage:
- Upload user profile pictures
- Store vehicle images
- Save announcement images
- Store any other visual assets

## Organization:
- Consider organizing images into subdirectories by category (e.g., `profiles/`, `vehicles/`, `announcements/`)
- Use descriptive filenames with timestamps or unique identifiers

## Security Notes:
- Always validate file types and sizes before uploading
- Consider implementing image optimization for web performance
